# A sweet REPL for OpenAI's GPT models

1. Create a configuration file (~/.config/gpt-repl.toml)

```toml
api_key = "FILL IN" # https://platform.openai.com/account/api-keys
max_tokens = 2048
model = "gpt-3.5-turbo"
temperature = 0.8
```
2. 

> cargo run
